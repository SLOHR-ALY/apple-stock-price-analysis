# Apple Stock Price Analysis using Machine Learning

## Overview
This project analyzes historical Apple stock price data using Python.
Linear Regression is applied to study trends and understand the limitations of simple predictive models.

## Dataset
Historical Apple stock price data (Apple Inc.).

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Key Learnings
- Time-series data analysis
- Linear Regression for trend analysis
- Model limitations in stock prediction
